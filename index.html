My Work

Intrusion Detection System for Smart Home Security

An Intrusion Detection System (IDS) for smart home security is a network security solution that monitors and analyzes network traffic to detect potential threats and alert homeowners. By identifying anomalies and malicious activity, IDS helps prevent unauthorized access to smart devices and protects personal data. Upon detecting a threat, the system sends alert messages and calls to notify homeowners, enabling them to take prompt action.

Smart Farming System Using Solar Power and Sensors With IoT

A Smart Farming System utilizes solar power and sensors with IoT technology to optimize crop yields and promote sustainable agriculture. By leveraging solar power, sensors monitor soil moisture, temperature, and humidity, while IoT platforms collect and analyze data for real-time monitoring and decision-making. This system increases efficiency, improves crop yields, and reduces waste, making it a valuable solution for modern farming.
